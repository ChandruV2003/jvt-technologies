# Public Launch Checklist

## Site

- finalize domain choice
- set the real contact email
- decide whether to add a booking link
- deploy the static site
- verify metadata, favicon, and social preview assets

## Mail And Identity

- create the mailbox
- configure SPF, DKIM, and DMARC
- finalize sender name and signature
- test inbound and outbound mail

## Demo

- record one 60 to 90 second live demo
- use the synthetic sample documents or an approved safe set
- capture one clean answer with citations on screen

## Lead Workflow

- import the first real lead list
- score fit before drafting anything
- generate drafts only for high-fit targets
- keep the first outreach wave small and reviewed

## Do Not Launch Until

- the placeholder email is gone
- the final domain is known
- there is a human-reviewed path for demo requests
- the demo video exists or the live-demo path is rehearsed

# Email And Domain Setup Checklist

Use this only when `JVT Technologies` is ready for real outbound and inbound demo requests.

## Domain

- register or confirm the primary domain
- set the canonical website host
- choose the public sender identity

## Mailbox

- choose a mailbox provider
- create a real sender mailbox such as `hello@...` or `founder@...`
- create a calendar-linked scheduling path if needed

## DNS Basics

- SPF
- DKIM
- DMARC
- MX records

## Sender Identity

- sender name
- business signature
- simple scheduling link if used
- reply-to behavior if different from sender

## Before Sending Anything

- test mailbox deliverability
- send to your own addresses first
- confirm the website contact details match the real mailbox
- keep sending volume extremely low at first
- do not enable any autonomous send loop
- verify the demo-request inbox is checked by a human
- add a simple booking path only after the mailbox is working

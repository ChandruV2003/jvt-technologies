# Client Workspace

This folder is a local-only client workspace scaffold.

Recommended usage:

- `00-admin`: engagement summary, agreement notes, contact info
- `00-admin/contracts`: signed agreements, SOWs, and redlines
- `01-intake`: intake notes, discovery material, scope definition
- `02-source-documents`: client-provided source material
- `03-working-files`: analysis, drafts, intermediate work
- `04-deliverables`: polished outputs for client delivery
- `05-meetings`: notes, agendas, recordings metadata
- `06-billing`: invoice notes, payment tracking support
- `07-archive`: completed and superseded material

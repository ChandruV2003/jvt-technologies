# Demo Video Narration Script

## Intro

JVT Technologies builds private AI systems for document-heavy businesses. This demo shows a private document assistant that can index internal documents, answer grounded questions, and cite the source passages behind each answer.

## Middle

Here we are uploading a small document set that looks like the kinds of policies and engagement materials many firms already manage internally.

Now we can ask an operational question, see a cited answer, and inspect the source snippets that support it. The point is not just speed. The point is having a private, reviewable workflow.

## Close

This can be deployed in a local-first or otherwise controlled environment, depending on the client’s privacy, infrastructure, and workflow needs.

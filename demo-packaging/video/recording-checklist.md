# Recording Checklist

- close unrelated windows and notifications
- start the backend locally before recording
- preload the JVT Technologies website in one tab
- preload the operator demo UI in another tab
- keep browser zoom high enough for citations to be readable
- use the synthetic sample documents
- ask two prepared questions only
- pause briefly before each click for easier editing later
- record one clean take before experimenting
- keep the answer mode visible on screen
- hold for one beat on the citation panel before moving on
